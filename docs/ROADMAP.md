# Roadmap

This document outlines planned features and improvements for Song of the Day.

## Planned Features

### High Priority

- **Lyrics fetching from Genius** - Automatically fetch and embed song lyrics in notes
- **Download album art to vault** - Store album art locally instead of using external URLs
  - Configurable storage location
  - Automatic image optimization

### Medium Priority

- **Batch processing for multiple songs** - Create notes for multiple tracks at once
  - Paste multiple Spotify URLs
  - Process entire playlists
- **Additional trigger methods**
  - Ribbon icon for quick access
  - Context menu integration

### Low Priority

- **Auto-discovery of related links**
  - Genius lyrics page
  - Wikipedia artist/album pages
  - AllMusic entries
- **Enhanced metadata**
  - Genre tags
  - Mood/energy metrics from Spotify
  - Popularity scores
- **Template system improvements**
  - Multiple template presets
  - Conditional template sections
  - Template variable extensions

## Future Considerations

- Integration with other music services (Apple Music, YouTube Music)
- Playlist management features
- Listening history tracking
